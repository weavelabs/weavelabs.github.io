Ruby on Rails

Ruby on Rails is a web framework in the Ruby language. With convention over configuration, and being in an easily readable language like Ruby, this framework is extensively used to build web apps in a fast pace.

React JS

React is a declarative, efficient, and flexible JavaScript library for building user interfaces. By building with the react ecosystem, your views can be made of responsive and reusable components which can talk to an API.

Ionic Applications
